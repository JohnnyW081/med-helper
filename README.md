# 用药助手

定时提醒服药，管理用药计划。纯前端单页应用，无需后端。

## 功能
- 多药品管理（名称、时间、剂量）
- 每日进度追踪
- 紧急联系人一键呼叫
- 用药提醒
- 多主题切换（纯黑/午夜青/日落橙/纯白）
- 数据本地存储（localStorage）

## 部署

### GitHub Pages
1. 创建GitHub仓库
2. 上传本目录所有文件
3. Settings → Pages → Source 选 main branch
4. 获取URL: `https://用户名.github.io/仓库名/`

### 本地运行
直接双击 `index.html` 即可

## 技术栈
- 纯HTML/CSS/JavaScript
- localStorage 数据持久化
- 响应式设计
- PWA 支持

# 用药助手 / Med Helper / お薬管理

定时提醒服药，管理用药计划。纯前端单页应用，无需后端。

## 功能
- 多药品管理（名称、时间、剂量）
- 每日进度追踪
- 紧急联系人一键呼叫
- 用药提醒推送（浏览器通知 + 震动）
- **3 种语言**：中文 / English / 日本語
- 4 种主题：午夜青 / 日落橙 / 纯黑 / 纯白
- 药品拍照上传
- 过期药物回收指南
- 数据本地存储（localStorage）
- PWA 离线支持

## 使用
直接双击 `index.html` 即可在浏览器中打开。设置页可切换语言和主题。

## 技术栈
- 纯 HTML/CSS/JavaScript
- localStorage 数据持久化
- 响应式移动端设计
- PWA 离线支持（Service Worker）
